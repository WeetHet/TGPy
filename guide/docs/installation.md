# Installation

Python 3.9+ required. Host TGPy and connect it to your Telegram account:

```shell
> pip install tgpy
> tgpy
```

Send `ping()` to any chat to check if TGPy is running.

## Updating

Update to the newest version with:

```shell
> pip install -U tgpy
```

Or right from Telegram:

```python
update()
```
